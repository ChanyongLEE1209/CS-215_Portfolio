# Peter Lee CS-215 Portfolio

## About Me

I am a student at Whitman College taking Data Science. This portfolio documents my final project for CS-215.

My final project explores how weather conditions affect daily productivity-related behavior. Since productivity is difficult to measure directly, I use physical activity data as a proxy for daily behavior.

## Project Focus

**Main Research Question:**

Do weather conditions affect low-, medium-, and high-activity participants differently?

This project uses a Norway wearable activity dataset from 2019–2020 and Open-Meteo weather data matched to the same date range.
